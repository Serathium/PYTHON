# Generating Requirements Text File  
To create a text file for the requirements of a project, in the activated project's environment, execute:  
`pip freeze > requirements.txt` 

# Installing Requirements From Another Project  
To install the requirements activate your project's environment and run:  
`pip install -r requirements.txt`  
If the file name isn't "requirements", use the appropriate name.  

# Verifying  
To verify installed packages, run:  
`pip list`
