# Creating Virtual Environment  
Inside the project's directory, execute:  
`python3 -m venv .venv`  
  
# Activating Environment  
Everytime the terminal is used to work on the project, execute (this needs to be ran inside the project's directory, unless an absolute path is provided):  
`source .venv/bin/activate` 
  
# Deactivating Environment  
To deactivate an environment, run:  
`deactivate`
